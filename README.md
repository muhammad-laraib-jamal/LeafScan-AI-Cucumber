# 🌿 LeafGuard AI  
### Smart Cucumber Leaf Disease Detection System

LeafGuard AI is a web-based application that uses Deep Learning to detect diseases in cucumber leaves.  
Upload an image of a leaf and get instant predictions along with confidence scores and treatment suggestions.

---

## 🚀 Features

- 📷 Upload cucumber leaf images
- 🤖 AI-based disease detection
- 📊 Confidence score display
- 🌿 Detects:
  - Healthy
  - Powdery Mildew
  - Leaf Spot
- 💡 Provides treatment suggestions
- 🎨 Modern UI (Glassmorphism + animations)

---

## 🧠 Tech Stack

- **Backend:** Flask (Python)
- **Machine Learning:** TensorFlow / Keras
- **Frontend:** HTML, CSS
- **Image Processing:** PIL, NumPy
